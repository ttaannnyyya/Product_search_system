AI tools were actively used throughout the development process. Initially, I used ChatGPT to discuss the overall approach, architecture, and design decisions for the application. Instead of directly generating code and using it blindly, I discussed multiple implementation approaches, asked follow-up questions, and evaluated the suggestions based on the assignment requirements. ChatGPT helped me refine the project structure, API design, database interactions, validation logic, and error-handling strategies. It also suggested several improvements that enhanced the overall quality and robustness of the application.

During implementation, I used GitHub Copilot as a pair-programming assistant for rapid development. I reviewed and understood the generated code before integrating it into the project. Whenever I encountered code that I did not fully understand, I analyzed it, asked questions, and verified its behavior through testing. I intentionally tested various edge cases such as empty inputs, duplicate product names, special characters, invalid API requests, and server unavailability. Whenever an issue was identified, I described the expected behavior to Copilot and refined the implementation accordingly. This AI-assisted workflow significantly reduced development time while still allowing me to maintain control over the design, logic, testing, and final implementation decisions.
